# Valentine's Day Page for Simphiwe                                            
                                                                                
 A fun, interactive "Will you be my Valentine?" webpage with a twist - the No   
 button runs away!                                                              
                                                                                
 ## Features                                                                    
                                                                                
 - Romantic red theme with glassmorphism design                                 
 - Floating hearts, rose petals, and sparkles                                   
 - The No button escapes when you try to click it                               
 - After 5 attempts, the button shrinks                                         
 - After 10 attempts, it becomes tiny and gives up                              
 - Confetti explosion when Yes is clicked                                       
 - Fully mobile responsive                                                      
 - Touch screen support                                                         
                                                                                
 ## Files                                                                       
                                                                                
 - `valentine.html` - Main HTML structure                                       
 - `valentine.css` - Styles and animations                                      
 - `valentine.js` - Interactive functionality                                   
                                                                                
 ## Usage                                                                       
                                                                                
 Open `valentine.html` in any modern web browser.                               
                                                                                
 ## Customization                                                               
                                                                                
 To change the recipient's name, edit `valentine.html` and replace "Simphiwe"   
 with your desired name.                                                        
                                                                                
 To change the message, edit the text inside the `<p class="message">` tag.
